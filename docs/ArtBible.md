# Prague 1600: Assassin Guild

# Art Bible v1.0

## Purpose

This document defines the visual identity of the project.

Every asset, level, character, AI-generated concept, and environment should follow these guidelines.

Goal:

Create a recognizable visual style that feels like Prague around 1600 while remaining achievable for a solo developer.

---

# Visual Pillars

## Pillar 1: Readability First

Gameplay clarity is more important than realism.

Players must immediately recognize:

- Guards
- Civilians
- Targets
- Entrances
- Restricted areas
- Hiding locations

If realism hurts readability, choose readability.

---

## Pillar 2: Atmosphere Through Simplicity

The game should feel:

- Mysterious
- Historical
- Slightly dangerous
- Intriguing

Atmosphere comes from:

- Lighting
- Composition
- Color palette

Not excessive detail.

---

## Pillar 3: Dense Lived-In Spaces

Locations should feel occupied.

Include:

- Crates
- Barrels
- Market stalls
- Benches
- Lanterns
- Carts

The city should feel alive.

---

# Camera

Perspective:

3/4 Top-Down

Inspired by:

- The Escapists
- Moonlighter
- Stardew Valley

Characteristics:

- Visible faces
- Visible building fronts
- Visible doors and windows

Avoid:

- Pure top-down
- True isometric

---

# Technical Standards

## Tile Size

Base Tile:

32x32

Why:

- Easy to work with
- Good detail level
- Strong Godot support

---

## Character Size

Target:

64x64

Minimum:

48x48

Reason:

Allows:

- Beards
- Hats
- Masks
- Uniform details

Characters must be recognizable.

# Lighting

Primary Light Sources

- Lanterns
- Candles
- Fireplaces

Goal:

Create stealth opportunities.

Avoid modern lighting aesthetics.

---

# Effects

Use sparingly.

Allowed:

- Candle flicker
- Smoke
- Rain
- Fog

Avoid:

- Excessive particles
- Flashy effects

---

# Animation Guidelines

Movement:

4-direction or 8-direction

Priorities:

1. Readability
2. Responsiveness
3. Personality

Not realism.

# Definition of Success

A screenshot should immediately communicate:

1. This is Prague.
2. This is around 1600.
3. This is a stealth game.
4. Guards, civilians, nobles, and alchemists are recognizable.
5. The world feels alive.
